# Python Lab 3 Solutions

This repository contains my solutions for Python programming exercises focused on strings and object-oriented programming concepts.

## Topics Covered

- String operations
- String slicing
- Splitting strings
- Repetition operator
- Login name generation
- Object-oriented programming (OOP)
- Classes and objects
- Constructors (`__init__`)
- Accessor and mutator methods
- Encapsulation
- UML-based class design

5. CellPhone class implementation

7. Customer class implementation

This lab was completed to practice string manipulation, object-oriented programming concepts, class design, and improve problem-solving skills using Python.
